1. What is html?

    2. HTML is the standard markup language for documents designed to be displayed in a web browser.It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

  2. What is difference b/w html & xhtml?

    3. HTML (HyperText Markup Language) and XHTML (Extensible HyperText Markup Language) are both markup languages used for creating web pages, but they differ primarily in their syntax strictness and underlying foundations.

  3. What is tag?

    4. HTML tags are the fundamental building blocks of web development, providing the structure and organization necessary for creating web pages. They are special words or characters enclosed in angle brackets < > that define the type of content on a web page and instruct web browsers on how to render and display that content.

  4. What is elements?

    5. An HTML element is a type of HTML (HyperText Markup Language) document component, one of several types of HTML nodes.The first used version of HTML was written by Tim Berners-Lee in 1993 and there have since been many versions of HTML. The current de facto standard is governed by the industry group WHATWG and is known as the HTML Living Standard.

  5. What is difference b/w html4 & html5?

    6. HTML5 significantly improves upon HTML4 with features like native multimedia support, semantic elements, and modern APIs, making it more powerful and easier to use. HTML5 also simplifies the document type declaration and enhances error handling compared to its predecessor.

  6. What are lists?

    7. An HTML List allows you to organize data on web pages into an ordered or unordered format to make the information easier to read and visually appealing. HTML Lists are very helpful for creating structured, accessible content in web development.There are three main types of lists in HTML:

    • Unordered Lists:These lists are used for items that do not need to be in any specific order. The list items are typically marked with bullets.
    • Ordered Lists: These lists are used when the order of the items is important. Each item in an ordered list is typically marked with numbers or letters.
    • Description Lists: These lists are used to contain terms and their corresponding descriptions.


  7. What is difference b/w XML & HTML?
    • HTML: Primarily used for structuring and displaying content on web pages. Its focus is on how data appears to the user in a browser
    • XML: Primarily used for storing and transporting data. Its focus is on describing the data itself, not its presentation.
